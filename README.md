### Description

A small program to change the Field of Vision in Resident Evil 7 biohazard.

### How it works
- Open the game and load a save
- Launch RE7FOV
- Adjust the slider to your desired Field of Vision

### Requirements
.NET Framework 4.7.2 or higher
